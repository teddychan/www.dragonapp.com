ClipMenu 2

macOS를 위한 빠른 클립보드 기록 메뉴 막대 앱.

질문하거나, 버그를 신고하거나, 기능을 요청하는 가장 빠른 방법은 공개 이슈 트래커입니다. 우리는 모든 신고를 읽습니다.

자주 묻는 질문

클립보드 기록은 어떻게 여나요?
전역 단축키(설정에서 지정)를 눌러 커서 위치에 기록을 띄우거나, 메뉴 막대의 ClipMenu 2 아이콘을 클릭하세요.
ClipMenu 2가 다른 앱에 붙여넣지 못합니다.
macOS는 사용자를 대신해 붙여넣으려면 손쉬운 사용 권한이 필요합니다. 시스템 설정 → 개인정보 보호 & 보안 → 손쉬운 사용을 열고 ClipMenu 2를 활성화하세요.
구독에는 무엇이 포함되나요?
Mac App Store 에디션은 30일 무료 체험 이후 자동 갱신되는 연간 구독이 적용되는 유료 앱입니다. 전체 앱과 모든 Mac 간 클립보드 기록 및 스니펫의 iCloud 동기화를 잠금 해제합니다.
새 Mac에서 구독을 어떻게 복원하나요?
동일한 Apple ID로 로그인한 상태에서 ClipMenu 2를 열고 시작 화면에서 구입 항목 복원을 선택하세요.
구독은 어떻게 관리하거나 취소하나요?
App Store 앱을 열고 → 이름을 클릭한 뒤 → 구독으로 이동하거나, Apple 구독 페이지를 방문하세요.
내 데이터는 어디에 저장되나요?
기록과 스니펫은 당신의 Mac과 본인의 비공개 iCloud 데이터베이스에 머뭅니다. 우리는 결코 그것을 받지 않습니다. 개인정보 처리방침을 참고하세요.

KeyKey

Yahoo!奇摩輸入法의 Cangjie(倉頡) & Simplex(速成) 입력기를 macOS에 맞게 새롭게 구축했습니다.

질문, 버그 신고, 기능 요청은 KeyKey 이슈 트래커로 보내 주세요. 하나하나 모두 읽고 있습니다.

GitHub에서 이슈 열기 →

이메일이 더 편하신가요? support@dragonapp.com으로 연락 주시면 답변드리겠습니다.

자주 묻는 질문

KeyKey는 어떻게 설치하나요?
최신 릴리스에서 YahooKeyKey2.pkg를 다운로드해 더블클릭하거나, brew install --cask teddychan/tap/yahoo-keykey-2를 실행하세요. 그런 다음 로그아웃했다가 다시 로그인하세요.
입력 소스에 나타나지 않습니다.
macOS는 로그인할 때만 입력기를 등록합니다. 로그아웃했다가 다시 로그인한 뒤 시스템 설정 → 키보드 → 입력 소스 → + → 번체 중국어에서 추가하고 倉頡(Cangjie) 및/또는 速成(Simplex)을 선택하세요.
KeyKey로 전환하고 설정을 변경하려면 어떻게 하나요?
Ctrl-Space를 눌러 입력 소스를 순환하세요. 메뉴 막대의 입력 메뉴에서 聯想字詞(연상 어구), 全形標點(전각 문장 부호), 輸出簡體字(간체자 출력)를 켜고 끄거나 關於(정보)를 열 수 있습니다.
간체 중국어는 어떻게 입력하나요?
입력 메뉴에서 輸出簡體字를 켜세요. 계속 Cangjie/Simplex로 입력하면 KeyKey가 간체자를 출력합니다.
실행하려면 무엇이 필요한가요?
macOS 12 이상을 실행하는 Apple Silicon 탑재 Mac이 필요합니다. KeyKey는 Developer ID로 서명되고 Apple의 공증을 받았으므로 문제없이 열립니다.
KeyKey는 어떻게 제거하나요?
brew uninstall --cask yahoo-keykey-2를 실행하거나 ~/Library/Input Methods/YahooKeyKey2.app를 삭제하세요. 그런 다음 입력 소스에서 제거하고 로그아웃했다가 다시 로그인하세요.
