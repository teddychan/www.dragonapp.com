macOS용 키보드 창 관리자

키 입력 한 번으로 창을 이동·크기 조절

Spectacle 2는 맨 앞 창을 절반·모서리·3분할·전체 화면으로 맞추고, 디스플레이 간 이동, 확대·축소, 실행 취소까지 — 모두 키보드로. macOS 26·27에서 실행되는 Spectacle의 현대적 오픈 소스 재구축입니다.

모든 창을 원하는 위치에 정확히

Spectacle의 창 동작 전체 — 모두 키보드로, 모두 재설정 가능.

절반과 전체 화면

⌥⌘와 화살표 키로 창을 왼쪽·오른쪽·위·아래 절반에 맞추거나 화면을 가득 채우세요.

모서리와 3분할

어느 사분면으로도 보내거나 ⌃⌥←와 ⌃⌥→로 화면의 여섯 3분할을 차례로 이동하세요.

다시 눌러 전환

절반이나 모서리 단축키를 반복해서 누르면 창이 ½, ⅔, ⅓로 전환됩니다 — Spectacle의 대표 기능.

디스플레이 간 이동

다음 또는 이전 디스플레이로 창을 보내면 가운데에 놓이고, 맞지 않으면 화면을 가득 채웁니다.

실행 취소와 다시 실행

마음에 안 드나요? ⌥⌘Z로 창을 원래 자리로 되돌리고 ⌥⌘⇧Z로 다시 이동합니다.

당신의 단축키, 무료이자 오픈

설정에서 모든 동작을 재설정하고, 클래식 기본값을 기본 제공하며, 전체 소스는 GitHub에 MIT 라이선스로 공개됩니다. 광고 없음, 추적 없음.

세 단계로 창 맞추기

Spectacle 2는 메뉴 막대에 상주하며 어디서나 작동합니다.

1

설치하고 접근 허용

GitHub 또는 Homebrew에서 Spectacle 2를 받아 실행하고, 창을 이동할 수 있도록 손쉬운 사용 권한을 허용하세요.

2

단축키 누르기

창이 활성화된 상태에서 왼쪽 절반은 ⌥⌘←, 가운데 정렬은 ⌥⌘C, 전체 화면은 ⌥⌘F 등을 누르세요.

3

전환과 취소

다시 눌러 ½ → ⅔ → ⅓ 전환, 3분할과 디스플레이를 차례로 이동, ⌥⌘Z로 어떤 이동이든 취소하세요.
