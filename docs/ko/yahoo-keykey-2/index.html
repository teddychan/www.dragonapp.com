되살아난 Yahoo! KeyKey 타이핑 경험

Mac 사용자들이 그리워하던 빠르고 막힘없는 Cangjie와 Simplex 입력 — Swift로 새롭게 구현했습니다.

倉頡 + 速成 입력

두 가지 클래식 모드 — 완전한 Cangjie (倉頡)와 Simplex (速成) — 그리고 모든 부수가 기억나지 않을 때 쓰는 와일드카드 * 매칭을 지원합니다.

빈도순 정렬 & 적응형

후보는 사용 빈도에 따라 정렬되며, 자주 선택하는 글자일수록 시간이 지나면서 상위로 올라옵니다 — 가장 많이 쓰는 단어가 맨 위에 표시됩니다.

연관 어휘 (聯想字詞)

글자를 확정하면 KeyKey가 보통 뒤따르는 단어를 제안합니다 — 자주 쓰는 어구를 한 번의 키 입력으로 완성할 수 있습니다.

繁 → 簡 & 전각 문장 부호

번체-간체 변환 출력(輸出簡體字)과 전각 문장 부호(全形標點)를 입력 메뉴에서 바로 전환하세요 — 별도의 환경설정 창이 필요 없습니다.

네이티브 후보 창

원본 레이아웃을 그대로 살린 세로 후보 목록 — 텍스트 커서를 따라다니며, 화면에 맞춰 위로 뒤집히거나 고정되어 잘리는 법이 없습니다.

가볍고 & 오픈소스

공유 리소스로 앱 전반의 메모리 사용량을 낮게 유지하며, 전체 소스가 GitHub에 공개되어 있습니다 — 무료, MIT 라이선스이며, 직접 살펴보거나 빌드할 수 있습니다.

네 단계면 입력 준비 완료

설치 프로그램은 Apple이 서명하고 공증해서 Gatekeeper 경고 없이 깔끔하게 열립니다.

1

다운로드 & 열기

GitHub 릴리스에서 YahooKeyKey2.pkg를 받아 더블 클릭해 설치하세요.

2

로그아웃

~/Library/Input Methods에 설치되며(관리자 권한 불필요), 이후 로그아웃 후 다시 로그인하라고 안내합니다.

3

입력 소스 추가

시스템 설정 ▸ 키보드 ▸ 입력 소스 ▸ + ▸ 번체 중국어 → 倉頡 및/또는 速成을 추가하세요.

4

전환 & 입력

Ctrl-Space를 눌러 KeyKey로 전환하고 입력을 시작하세요. 입력 메뉴에서 簡體를 전환하거나 關於를 열 수 있습니다.

헌정

KeyKey는 Swift로 만든 독립적인 오픈소스 재구현으로, 많은 Mac 사용자가 사랑했던 번체 중국어 입력기 원작 Yahoo! KeyKey (Yahoo!奇摩輸入法)에 대한 헌정으로 제작되었습니다. 그 경험을 현대 macOS로 이어가며 MIT License로 배포됩니다.
