macOS를 위한 메뉴 막대 관리 앱

메뉴 막대를 길들이세요 — 현대 macOS에 돌아왔습니다

Ice는 필요 없는 메뉴 막대 항목을 숨기고, 원할 때 다시 불러오며, 메뉴 막대를 원하는 대로 배치할 수 있게 해줍니다. 충실한 오픈소스 재구현 — 이제 macOS 26과 27에서 실행됩니다.

더 차분한 메뉴 막대, 원하는 대로

메뉴 막대를 정돈하고, 배치하고, 다시 보여주는 데 필요한 모든 것 — 그리고 불필요한 것은 없습니다.

메뉴 막대 항목 숨기기

거의 쓰지 않는 아이콘을 구분선 뒤에 감춰, 메뉴 막대를 깔끔하고 집중력 있게 유지하세요.

Ice Bar

숨긴 항목을 별도의 막대로 옮겨, 부를 때만 나타나게 하세요 — 어수선함 없이, 잃어버리는 것도 없습니다.

필요할 때 보기

클릭, 전역 단축키, 또는 가장자리에 마우스를 올리는 동작으로 숨긴 항목을 펼치세요 — 원하는 방식으로.

배치 & 간격 조절

항목 순서를 바꾸고 간격을 조절해, 메뉴 막대를 원하는 그대로 만드세요.

맞춤 단축키 & 모양

표시 여부를 토글하는 단축키를 지정하고, 메뉴 막대의 색조, 그림자, 테두리를 취향에 맞게 조정하세요.

무료 & 오픈소스

전체 소스가 GPLv3 하에 GitHub에 공개되어 있어 — 자유롭게 살펴보고, 빌드하고, 신뢰할 수 있습니다. 광고도, 추적도 없습니다.

세 단계면 정돈 완료

Ice는 메뉴 막대에 머물며 당신의 작업을 방해하지 않습니다.

1

설치 & 실행

GitHub 또는 Homebrew에서 Ice를 받아 열면 — 메뉴 막대에 조용히 자리 잡습니다.

2

숨길 항목 선택

⌘-드래그로 구분선을 옮겨, 그 왼쪽에 있는 항목을 필요할 때까지 숨겨 두세요.

3

언제든 펼치기

Ice 아이콘을 클릭하거나 단축키를 눌러 전부 보여준 뒤, 다시 자동으로 숨겨지게 하세요.

Ice를 받는 두 가지 방법

똑같은 무료 오픈소스 앱입니다. 당신의 작업 방식에 맞는 것을 고르세요.
