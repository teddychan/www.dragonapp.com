많이 복사하는 사람을 위해 만들었습니다

Mac에서 텍스트와 이미지를 이리저리 옮기는 데 필요한 모든 것, 그리고 불필요한 것은 없습니다.

클립보드 기록

모든 복사가 기억됩니다. 최근 클립을 커서 위치나 메뉴 막대에서 바로 띄워 단축키 하나로 붙여넣으세요.

폴더 속 스니펫

늘 입력하는 텍스트 — 답장, 주소, 코드 — 를 저장하고, 직접 관리하는 폴더로 정리하세요.

JavaScript 액션

클립을 즉석에서 변환하세요: 대소문자 변경, 공백 제거, HTML 또는 Base64 인코딩, 해시, 일본어 변환 등.

전역 단축키

직접 지정한 단축키로 메뉴를 열고 맨 앞 앱에 바로 붙여넣으세요. 손이 키보드를 떠날 일이 없습니다.

이미지 썸네일

이미지를 복사했나요? 인라인 썸네일과 번호가 매겨진 항목, 툴팁으로 항상 올바른 것을 고를 수 있습니다.

iCloud 동기화

App Store 에디션에서는 스니펫과 설정이 본인의 비공개 iCloud를 통해 모든 Mac에 동기화됩니다.

세 단계, 막힘 없이

ClipMenu 2는 메뉴 막대에서 조용히 머물며 당신의 작업을 방해하지 않습니다.

커서 위치에 기록이 팝업됩니다. 화살표나 번호로 클립을 고르세요.

어디에나 붙여넣기

궁금증 해결

App Store 버전과 GitHub 버전의 차이는 무엇인가요?

같은 앱입니다. Mac App Store 에디션은 iCloud 동기화를 추가하고 App Store를 통해 자동으로 업데이트되는 유료 구독입니다. GitHub 에디션은 무료 오픈소스로, 내장 자동 업데이트를 제공하며 iCloud 동기화를 제외한 모든 기능을 포함합니다.

ClipMenu 2는 왜 손쉬운 사용 권한을 요청하나요?

macOS에서는 앱이 사용자를 대신해 다른 앱에 붙여넣으려면 손쉬운 사용 권한이 필요합니다. ClipMenu 2는 선택한 클립을 맨 앞 앱에 넣을 때만 이 권한을 사용합니다. 시스템 설정 → 개인정보 보호 & 보안 → 손쉬운 사용에서 권한을 부여하세요. (App Store 에디션은 샌드박싱되어 이 권한을 사용하지 않습니다.)

내 클립보드 데이터는 어디에 저장되나요?

당신의 Mac에 저장됩니다. 기록과 스니펫은 절대 기기를 떠나지 않으며 — ClipMenu 2에는 분석, 광고, 추적이 없고, 우리는 결코 당신의 클립보드 내용을 받지 않습니다. iCloud 동기화를 켜면 스니펫이 본인의 비공개 iCloud 계정을 통해 전송됩니다. 개인정보 처리방침을 읽어보세요.

비용은 얼마인가요?

GitHub 빌드는 영원히 무료입니다. Mac App Store 에디션은 30일 무료 체험 이후 연 $9.99이며, 여기에는 iCloud 동기화와 App Store 업데이트가 포함됩니다. Apple 계정 → 구독에서 언제든지 관리하거나 취소할 수 있습니다.

실행하려면 무엇이 필요한가요?

macOS 26 이상을 실행하는 Mac이 필요합니다. ClipMenu 2는 Dock 아이콘 없이 메뉴 막대 에이전트로 실행되며 당신의 작업을 방해하지 않습니다.

이것이 원작 ClipMenu인가요?

ClipMenu 2는 Naotaka Morimoto의 원작 ClipMenu의 유산 위에 만든, Swift로 작성한 독립적이고 현대적인 재작성판입니다. MIT License 하의 오픈소스인 그의 작업에 깊이 감사하며 — ClipMenu 2는 그 정신을 이어받아 동일한 라이선스로 배포됩니다.
