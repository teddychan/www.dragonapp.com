よくある質問

クリップボード履歴はどうやって開きますか？
グローバルホットキー（設定で指定）を押すと履歴がカーソル位置にポップアップします。または、メニューバーのClipMenu 2アイコンをクリックします。
ClipMenu 2が他のアプリに貼り付けられません。
macOSでは、ユーザーに代わって貼り付けるためにアクセシビリティ権限が必要です。システム設定 → プライバシーとセキュリティ → アクセシビリティを開き、ClipMenu 2を有効にしてください。
サブスクリプションには何が含まれますか？
Mac App Store版は30日間の無料トライアル付きの有料アプリで、その後は年単位で自動更新されるサブスクリプションになります。アプリのフル機能と、すべてのMac間でのクリップボード履歴・スニペットのiCloud同期が利用できます。
新しいMacでサブスクリプションを復元するには？
同じApple IDでサインインした状態でClipMenu 2を開き、ウェルカム画面で購入を復元を選んでください。
サブスクリプションを管理・解約するには？
App Storeアプリを開き → 自分の名前をクリック → サブスクリプション、またはAppleのサブスクリプションにアクセスしてください。
データはどこに保存されますか？
履歴とスニペットはあなたのMac上と、ご自身のプライベートなiCloudデータベースに留まります。私たちが受け取ることはありません。プライバシーポリシーをご覧ください。

よくある質問

KeyKey はどうやってインストールしますか？
最新リリースから YahooKeyKey2.pkg をダウンロードしてダブルクリックするか、brew install --cask teddychan/tap/yahoo-keykey-2 を実行してください。その後、一度ログアウトして再度ログインしてください。
入力ソースに表示されません。
macOS は入力方式をログイン時にのみ登録します。一度ログアウトして再度ログインし、システム設定 → キーボード → 入力ソース → + → 繁体字中国語から追加して、倉頡（Cangjie）や速成（Simplex）を選択してください。
KeyKey への切り替えや設定の変更はどうしますか？
Ctrl-Space を押すと入力ソースを切り替えられます。メニューバーの入力メニューから、聯想字詞（連想語句）、全形標點（全角の句読点）、輸出簡體字（簡体字での出力）を切り替えたり、關於（このアプリについて）を開いたりできます。
簡体字中国語を入力するにはどうしますか？
入力メニューで 輸出簡體字 をオンにしてください。倉頡／速成での入力はそのままに、KeyKey が簡体字を出力します。
動作には何が必要ですか？
Apple Silicon を搭載し、macOS 12 以降を実行している Mac が必要です。KeyKey は Developer ID で署名され、Apple による公証を受けているため、問題なく起動できます。
KeyKey はどうやってアンインストールしますか？
brew uninstall --cask yahoo-keykey-2 を実行するか、~/Library/Input Methods/YahooKeyKey2.app を削除してください。その後、入力ソースから削除し、一度ログアウトして再度ログインしてください。

Ice 2

macOS向けメニューバーマネージャー — メニューバーのアイコンを非表示にして整理し、必要なときに表示できます。

ご質問、バグ報告、機能のご要望は Ice 2 のイシュートラッカーへお寄せください。すべて拝見しています。

GitHubでイシューを開く →

メールがよろしいですか？support@dragonapp.comまでご連絡ください。折り返しご返信します。

よくある質問

Ice 2 はどうやってインストールしますか？
最新リリースからダウンロードしてアプリケーションフォルダにドラッグするか、brew install --cask teddychan/tap/ice-2 を実行してください。
メニューバーのアイコンを非表示にしたり表示したりするには？
⌘ キーを押しながら仕切りをドラッグして、左側のアイコンが非表示になるよう配置します。Ice 2 アイコンをクリックするか、ホットキーを押すか、エッジにホバーすることで再び表示できます。
Ice 2 の動作に必要なものは？
macOS 26 以降が動作する Mac です。
Ice 2 はどうやってアンインストールしますか？
brew uninstall --cask ice-2 を実行するか、Ice 2 を終了してアプリケーションフォルダからゴミ箱にドラッグしてください。
