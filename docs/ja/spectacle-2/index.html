すべてのウインドウを思いどおりの場所に

Spectacle のウインドウ操作をすべて網羅 — すべてキーボードで、すべて再割り当て可能。

半分とフルスクリーン

⌥⌘ と矢印キーで、ウインドウを左・右・上・下の半分に配置、または画面いっぱいに。

四隅と三分割

任意の四分の一に配置、または ⌃⌥← と ⌃⌥→ で画面の 6 つの三分割を順に切り替え。

もう一度押して切り替え

半分や四隅のショートカットを繰り返し押すと、ウインドウが ½・⅔・⅓ と切り替わる — Spectacle 定番の操作。

ディスプレイ間で移動

次または前のディスプレイへウインドウを送ると中央に配置 — 収まらない場合は画面いっぱいに。

取り消しとやり直し

しっくりこない？ ⌥⌘Z で元の位置に戻し、⌥⌘⇧Z でもう一度動かせます。

あなたのショートカット、無料でオープン

すべての操作を設定で再割り当て、従来のデフォルトも搭載。全ソースは GitHub に MIT ライセンスで公開。広告なし・追跡なし。

3 ステップでウインドウを配置

Spectacle 2 はメニューバーに常駐し、どこでも動作します。

1

インストールしてアクセスを許可

GitHub または Homebrew から Spectacle 2 を入手して起動し、ウインドウを動かせるようアクセシビリティ権限を許可します。

2

ショートカットを押す

ウインドウを選択した状態で、左半分は ⌥⌘←、中央寄せは ⌥⌘C、フルスクリーンは ⌥⌘F など。

3

切り替えと取り消し

もう一度押して ½ → ⅔ → ⅓、三分割やディスプレイを順に切り替え、⌥⌘Z でどの移動も取り消し。
