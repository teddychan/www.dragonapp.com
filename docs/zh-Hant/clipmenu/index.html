問題，一一解答

App Store 版與 GitHub 版有甚麼分別？

它們是同一個應用程式。Mac App Store 版本是付費訂閱，加入了 iCloud 同步，並透過 App Store 自動更新。GitHub 版本免費而開源，內建自動更新，並具備除 iCloud 同步以外的每項功能。

為甚麼 ClipMenu 2 要求輔助使用權限？

macOS 規定應用程式須有輔助使用權限，才能代你貼入另一個應用程式。ClipMenu 2 只用它把你選定的片段放進最前的應用程式。請在 系統設定 → 私隱權與安全性 → 輔助使用 中授權。（App Store 版本已沙盒化，不會用到此項。）

我的剪貼簿資料儲存在哪裡？

儲存在你的 Mac 上。你的記錄與片段絕不會離開你的裝置 — ClipMenu 2 沒有分析、廣告或追蹤，我們亦絕不會接收你的剪貼簿內容。開啟 iCloud 同步後，片段會透過你自己的私人 iCloud 帳戶傳輸。請閱讀 私隱政策。

收費多少？

GitHub 版本永久免費。Mac App Store 版本提供 30 日免費試用，其後每年 $9.99，涵蓋 iCloud 同步及 App Store 更新。可隨時在你的 Apple 帳戶 → 訂閱項目 管理或取消。

執行它需要甚麼？

一部執行 macOS 26 或以上的 Mac。ClipMenu 2 以選單列代理程式形式運行，沒有 Dock 圖示，絕不礙事。

這是原版 ClipMenu 嗎？

ClipMenu 2 是一款以 Swift 打造的獨立現代重寫版，建基於 Naotaka Morimoto 原版 ClipMenu 的傳承。我們感激他的成果，那是以 MIT 授權開源的 — ClipMenu 2 把那份精神延續下去，並以同樣的授權發佈。
