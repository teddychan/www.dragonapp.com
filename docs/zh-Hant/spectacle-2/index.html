適用於 macOS 的鍵盤視窗管理器

一個按鍵，移動和調整視窗

Spectacle 2 把最前面的視窗貼齊到半螢幕、邊角、三分之一或全螢幕，跨顯示器移動、放大或縮小，還能還原——全部用鍵盤完成。這是 Spectacle 的現代開源重製版，現已支援 macOS 26 與 27。

半螢幕與全螢幕

用 ⌥⌘ 和方向鍵，把視窗貼齊到左、右、上、下半螢幕，或鋪滿整個螢幕。

邊角與三分之一

送到任意四分之一，或用 ⌃⌥← 和 ⌃⌥→ 在螢幕的六個三分之一之間切換。

跨顯示器

把視窗丟到下一個或上一個顯示器，它會置中落下——放不下時則鋪滿螢幕。
