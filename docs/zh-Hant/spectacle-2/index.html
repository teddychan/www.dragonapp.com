每個視窗，都在你想要的位置

Spectacle 的全套視窗動作——全部用鍵盤，全部可重新綁定。

半螢幕與全螢幕

用 ⌥⌘ 和方向鍵，把視窗貼齊到左、右、上、下半螢幕，或鋪滿整個螢幕。

邊角與三分之一

送到任意四分之一，或用 ⌃⌥← 和 ⌃⌥→ 在螢幕的六個三分之一之間切換。

再次按下切換

反覆按半螢幕或邊角快速鍵，視窗會在 ½、⅔ 和 ⅓ 之間切換——Spectacle 的標誌性操作。

跨顯示器

把視窗丟到下一個或上一個顯示器，它會置中落下——放不下時則鋪滿螢幕。

還原與重做

不太對？⌥⌘Z 把視窗放回原位，⌥⌘⇧Z 再次移動。

你的快速鍵，免費且開源

在設定中重新綁定每個動作，內建經典預設值，全部原始碼在 GitHub 上以 MIT 授權公開。無廣告，無追蹤。

三步貼齊視窗

Spectacle 2 常駐選單列，隨處可用。

1

安裝並授予存取權

從 GitHub 或 Homebrew 取得 Spectacle 2，打開它，並授予「輔助使用」權限以便移動視窗。

2

按下快速鍵

在任意視窗位於前景時，按 ⌥⌘← 貼左半螢幕，⌥⌘C 置中，⌥⌘F 全螢幕，等等。

3

切換與還原

再次按下切換 ½ → ⅔ → ⅓，在三分之一和顯示器之間切換，⌥⌘Z 可還原任意移動。
