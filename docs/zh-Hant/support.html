常見問題

如何開啟我的剪貼簿記錄？
按你的全域快速鍵（在設定中指定），即可在游標處彈出記錄，或按選單列上的 ClipMenu 2 圖示。
ClipMenu 2 無法貼入其他應用程式。
macOS 規定須有輔助使用權限才能代你貼上。請開啟 系統設定 → 私隱權與安全性 → 輔助使用，並啟用 ClipMenu 2。
訂閱包含甚麼？
Mac App Store 版本是付費應用程式，提供 30 日免費試用，其後為自動續期的年度訂閱。它會解鎖完整應用程式，以及在你所有 Mac 之間同步剪貼簿記錄與片段的 iCloud 同步。
如何在新的 Mac 上回復我的訂閱？
開啟 ClipMenu 2，在歡迎畫面選擇 回復購買項目，並以相同的 Apple ID 登入。
如何管理或取消訂閱？
開啟 App Store 應用程式 → 按你的名稱 → 訂閱項目，或前往 你的 Apple 訂閱項目。
我的資料儲存在哪裡？
你的記錄與片段會保留在你的 Mac 及你自己的私人 iCloud 資料庫中。我們絕不會接收。請參閱我們的 私隱政策。

常見問題

點樣安裝 KeyKey？
由最新版本下載 YahooKeyKey2.pkg 並雙擊安裝，或者執行 brew install --cask teddychan/tap/yahoo-keykey-2。之後登出再重新登入即可。
輸入來源裡面冇顯示出嚟。
macOS 只會喺登入時註冊輸入法。請先登出再重新登入，然後喺 系統設定 → 鍵盤 → 輸入來源 → + → 繁體中文 加入，並選擇 倉頡 (Cangjie) 同／或 速成 (Simplex)。
點樣切換去 KeyKey 同更改設定？
㩒 Ctrl-Space 可以循環切換輸入來源。喺選單列嘅輸入選單裡面，你可以開關 聯想字詞 (associated phrases)、全形標點 (full-width punctuation) 同 輸出簡體字 (Simplified output)，又或者打開 關於 (About)。
點樣輸入簡體中文？
喺輸入選單裡面開啟 輸出簡體字。你照樣用倉頡／速成輸入，KeyKey 就會輸出簡體字。
執行 KeyKey 需要乜嘢？
一部 Apple Silicon、執行 macOS 12 或以上版本嘅 Mac。KeyKey 已用 Developer ID 簽署並經 Apple 公證，所以可以順利開啟。
點樣解除安裝 KeyKey？
執行 brew uninstall --cask yahoo-keykey-2，或者刪除 ~/Library/Input Methods/YahooKeyKey2.app。之後喺輸入來源裡面移除佢，再登出並重新登入。
