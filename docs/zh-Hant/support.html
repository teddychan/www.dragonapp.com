常見問題

如何開啟我的剪貼簿記錄？
按你的全域快速鍵（在設定中指定），即可在游標處彈出記錄，或按選單列上的 ClipMenu 2 圖示。
ClipMenu 2 無法貼入其他應用程式。
macOS 規定須有輔助使用權限才能代你貼上。請開啟 系統設定 → 私隱權與安全性 → 輔助使用，並啟用 ClipMenu 2。
訂閱包含甚麼？
Mac App Store 版本是付費應用程式，提供 30 日免費試用，其後為自動續期的年度訂閱。它會解鎖完整應用程式，以及在你所有 Mac 之間同步剪貼簿記錄與片段的 iCloud 同步。
如何在新的 Mac 上回復我的訂閱？
開啟 ClipMenu 2，在歡迎畫面選擇 回復購買項目，並以相同的 Apple ID 登入。
如何管理或取消訂閱？
開啟 App Store 應用程式 → 按你的名稱 → 訂閱項目，或前往 你的 Apple 訂閱項目。
我的資料儲存在哪裡？
你的記錄與片段會保留在你的 Mac 及你自己的私人 iCloud 資料庫中。我們絕不會接收。請參閱我們的 私隱政策。

常見問題

How do I install KeyKey?
Download YahooKeyKey2.pkg from the latest release and double-click it, or run brew install --cask teddychan/tap/yahoo-keykey-2. Then log out and back in.
It doesn't show up in Input Sources.
macOS only registers an input method at login. Log out and back in, then add it under System Settings → Keyboard → Input Sources → + → Traditional Chinese and choose 倉頡 (Cangjie) and/or 速成 (Simplex).
How do I switch to KeyKey and change settings?
Press Ctrl-Space to cycle input sources. From the input menu in the menu bar you can toggle 聯想字詞 (associated phrases), 全形標點 (full-width punctuation), and 輸出簡體字 (Simplified output), or open 關於 (About).
How do I type Simplified Chinese?
Turn on 輸出簡體字 in the input menu. You keep typing Cangjie/Simplex and KeyKey outputs Simplified characters.
What do I need to run it?
A Mac with Apple Silicon running macOS 12 or later. KeyKey is signed with a Developer ID and notarized by Apple, so it opens cleanly.
How do I uninstall KeyKey?
Run brew uninstall --cask yahoo-keykey-2, or delete ~/Library/Input Methods/YahooKeyKey2.app. Then remove it from Input Sources and log out and back in.
