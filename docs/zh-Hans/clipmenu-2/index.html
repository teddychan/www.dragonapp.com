剪贴板历史

每一次复制都被记住。在光标处或菜单栏中弹出近期剪贴项，一个按键即可粘贴。

文件夹中的片段

保存你经常输入的文字——回复、地址、代码——并整理进由你掌控的文件夹。

JavaScript 动作

即时转换剪贴内容：改变大小写、修剪、编码 HTML 或 Base64、哈希、转换日文等等。
