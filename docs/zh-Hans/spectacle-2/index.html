面向 macOS 的键盘窗口管理器

一个按键，移动和调整窗口

Spectacle 2 把最前面的窗口贴靠到半屏、边角、三分之一或全屏，跨显示器移动、放大或缩小，还能撤销——全部用键盘完成。这是 Spectacle 的现代开源重制版，现已支持 macOS 26 与 27。

每个窗口，都在你想要的位置

Spectacle 的全套窗口操作——全部用键盘，全部可重新绑定。

半屏与全屏

用 ⌥⌘ 和方向键，把窗口贴靠到左、右、上、下半屏，或铺满整个屏幕。

边角与三分之一

发送到任意四分之一，或用 ⌃⌥← 和 ⌃⌥→ 在屏幕的六个三分之一之间切换。

再次按下切换

反复按半屏或边角快捷键，窗口会在 ½、⅔ 和 ⅓ 之间切换——Spectacle 的标志性操作。

跨显示器

把窗口甩到下一个或上一个显示器，它会居中落下——放不下时则铺满屏幕。

撤销与重做

不太对？⌥⌘Z 把窗口放回原位，⌥⌘⇧Z 再次移动。

你的快捷键，免费且开源

在设置中重新绑定每个操作，内置经典默认值，全部源代码在 GitHub 上以 MIT 许可证公开。无广告，无追踪。

三步贴靠窗口

Spectacle 2 常驻菜单栏，随处可用。

1

安装并授予访问权限

从 GitHub 或 Homebrew 获取 Spectacle 2，打开它，并授予「辅助功能」权限以便移动窗口。

2

按下快捷键

在任意窗口处于前台时，按 ⌥⌘← 贴左半屏，⌥⌘C 居中，⌥⌘F 全屏，等等。

3

切换与撤销

再次按下切换 ½ → ⅔ → ⅓，在三分之一和显示器之间切换，⌥⌘Z 可撤销任意移动。
