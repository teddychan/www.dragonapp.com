面向 macOS 的键盘窗口管理器

一个按键，移动和调整窗口

Spectacle 2 把最前面的窗口贴靠到半屏、边角、三分之一或全屏，跨显示器移动、放大或缩小，还能撤销——全部用键盘完成。这是 Spectacle 的现代开源重制版，现已支持 macOS 26 与 27。

功能演示

一个按键，移动和调整窗口
⌥ ⇧ ◔ ⤓ 9:41

半屏与全屏

用 ⌥⌘ 和方向键，把窗口贴靠到左、右、上、下半屏，或铺满整个屏幕。

边角与三分之一

发送到任意四分之一，或用 ⌃⌥← 和 ⌃⌥→ 在屏幕的六个三分之一之间切换。

跨显示器

把窗口甩到下一个或上一个显示器，它会居中落下——放不下时则铺满屏幕。
