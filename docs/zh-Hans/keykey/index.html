重现 Yahoo! KeyKey 的输入体验

Mac 用户念念不忘的快速、顺滑的倉頡与速成输入——用 Swift 全新打造。

倉頡 + 速成 输入

两种经典模式——完整的倉頡（Cangjie）与速成（Simplex）——并支持通配符 * 匹配，让你不必记清每个部首。

按词频排序 & 自适应

候选字按常用程度排列，你选过的字会随时间排得更靠前——让最常用的字词自然浮到顶部。

联想字词（聯想字詞）

在你确认一个字之后，KeyKey 会推荐通常紧随其后的词——让常用短语一次按键即可成词。

繁 → 簡 & 全角标点

直接从输入菜单切换繁转简输出（輸出簡體字）与全角标点（全形標點）——无需另开偏好设置窗口。

原生候选窗口

贴合原版布局的纵向候选列表——它会跟随文字光标，并在屏幕边缘上翻或贴边，绝不被遮挡。

轻量 & 开源

共享资源让各应用之间的内存占用保持低位，完整源代码就在 GitHub 上——免费、采用 MIT 许可，任你查阅或自行构建。

四步即可开始输入

安装包由 Apple 签名并公证，可干净打开——没有 Gatekeeper 警告。

1

下载 & 打开

从 GitHub 发布页获取 YahooKeyKey2.pkg，双击即可安装。

2

注销账户

它会安装到 ~/Library/Input Methods（无需管理员权限），然后提示你注销并重新登录。

3

添加输入源

系统设置 ▸ 键盘 ▸ 输入法 ▸ + ▸ 繁体中文 → 添加 倉頡 和／或 速成。

4

切换 & 输入

按 Ctrl-Space 切换到 KeyKey 并开始输入。可从输入菜单切换 簡體 或打开 關於。
