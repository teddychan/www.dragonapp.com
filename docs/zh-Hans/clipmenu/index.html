问题，已解答

App Store 版本与 GitHub 版本有什么区别？

它们是同一款应用。Mac App Store 版本是一份付费订阅，额外提供 iCloud 同步，并通过 App Store 自动更新。GitHub 版本免费且开源，内置自动更新，包含除 iCloud 同步以外的每一项功能。

ClipMenu 2 为什么要请求辅助功能权限？

macOS 要求应用获得辅助功能权限，才能代你粘贴到另一个应用中。ClipMenu 2 仅用它把你选中的剪贴项放进最前方的应用。请在 系统设置 → 隐私与安全性 → 辅助功能 下授予该权限。（App Store 版本已沙盒化，不使用此权限。）

我的剪贴板数据存储在哪里？

存储在你的 Mac 上。你的历史和片段绝不会离开你的设备——ClipMenu 2 没有任何分析、广告或追踪，我们也绝不接收你的剪贴板内容。开启 iCloud 同步后，片段会通过你自己的私密 iCloud 账户传输。请阅读隐私政策。

它要多少钱？

GitHub 版本永久免费。Mac App Store 版本为 30 天免费试用，之后每年 $9.99，费用涵盖 iCloud 同步与 App Store 更新。可随时在你的 Apple 账户 → 订阅 中管理或取消。

运行它需要什么条件？

一台运行 macOS 26 或更高版本的 Mac。ClipMenu 2 以菜单栏代理形式运行，没有程序坞图标，绝不碍事。

这是原版的 ClipMenu 吗？

ClipMenu 2 是一款用 Swift 编写的独立现代重写作品，承袭 Naotaka Morimoto 所作原版 ClipMenu 的传承。我们感激他的工作，其作品依据 MIT 许可协议开源——ClipMenu 2 将这份精神延续下去，并以同样的许可协议发布。
