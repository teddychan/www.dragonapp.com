macOS 上的菜单栏管理工具

整理你的菜单栏——回归现代 macOS

Ice 2 隐藏你不需要的菜单栏图标，在你需要时将它们召回，并让你自由编排菜单栏布局。忠于原作的开源重塑——现已运行在 macOS 26 和 27 上。
